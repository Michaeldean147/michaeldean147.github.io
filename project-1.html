ChowSpana

Project Description

My personal "capstone" project for gSchool. Ruby on Rails app that utilizes Javascript and jQuery. An app for 'foodies' who are always on the hunt for odd or rare food items. Users sign-up/sign-in and are then able to post their interesting food finds. This includes the name, price, location, address, along with a description and photo of the item. Other users can then view their posts and favorite the ones they like. The site uses Javascript and jQuery to handle the sorting of favorites and to allow users to view a static map image of the items location.
